# The Forsaken of Gambling
</p>
<p align="center">
## This mod was made as a passion project. I don't work for the Forsaken Development Team; definitely never will with what's been brewing.
</p>
The Forsaken of Gambling (TFoG) is a mod that adds tons of new joker ideas to Balatro.

TFOG adds the following:
- 20+ Jokers
- 1 Booster Pack
- 1 Deck

Cross-mod support:
- [Talisman](https://github.com/SpectralPack/Talisman)
